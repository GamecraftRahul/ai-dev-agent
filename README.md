# AI-Powered Software Development Agent

## Overview  
This project is an AI-powered software development agent capable of designing and generating full-stack applications from scratch. It follows strict rules, creates complete code files, and ensures code quality.

## Features  
- Full-stack application design and generation  
- File creation with complete code blocks  
- Strict code quality assurance  

## Requirements  
- Python 3.x  
- `crewai` library (installation instructions below)  

## Installation  
To install the required library, run:  
```bash
pip install crewai
